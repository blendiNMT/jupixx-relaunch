# Jupixx Relaunch 2025

> Modernisierter Website-Relaunch für [Jupixx](https://www.jupixx.at/) – die digitale Schneiderei aus Österreich für personalisierte Laserprodukte aus Holz.

## 🎨 Über das Projekt

Dieser Relaunch modernisiert die bestehende Jupixx-Website mit einem warmen, handgemachten Design, das die Werte der Marke perfekt widerspiegelt: persönlich, hochwertig, mit Liebe gefertigt.

## 🎯 Zielgruppe (MasterAvatar)

- **Anna, 34 Jahre**, Mutter aus Österreich/DACH
- Wertschätzt Handgemachtes und Personalisierung
- Sucht besondere Geschenke für Familie und Freunde
- Möchte Erinnerungen (z. B. Kinderzeichnungen) verewigen
- Aktiv auf Instagram & Pinterest

## 🎨 Design-System

### Farben
- **Warmes Holzbraun:** `#3D2817`
- **Creme:** `#F5EFE6`
- **Terrakotta (Akzent):** `#C97B5C`
- **Salbei:** `#7D8B6A`
- **Hintergrund:** `#FAF6F0`

### Typografie
- **Display:** Playfair Display (Serif)
- **Body:** Inter (Sans-Serif)
- **Handschrift:** Caveat

## 📄 Seiten

1. **index.html** – Startseite mit Hero, Process, Bestseller, Story, Testimonials, CTA
2. **shop.html** – Produktkatalog mit Filter und 12 Produktkarten
3. **personalisierung.html** – 5-Schritte-Prozess + Tipps
4. **ueber-uns.html** – Markengeschichte, Werte, Zahlen
5. **faq.html** – Interaktives Accordion mit 12 Fragen
6. **kontakt.html** – Kontaktformular + Kanäle

## ✨ Features

- 100% responsive (Mobile-first)
- CSS-Animationen & Hover-Effekte
- Scroll-Reveal Animationen
- Parallax-Effekt im Hero
- Funktionierender Produkt-Filter
- FAQ-Accordion
- Mobile Navigation
- Animierte Count-Up-Stats
- Keine externen JS-Frameworks – pure HTML/CSS/JS
- SVG-Illustrationen (keine externen Bilder nötig)

## 🚀 Tech-Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display, Inter, Caveat)

## 📦 Deployment

Die Seite ist als statische Website konzipiert und kann direkt auf Vercel, Netlify oder GitHub Pages gehostet werden.

```bash
# Lokal testen
npx serve .
```
